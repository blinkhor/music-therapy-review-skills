# 音乐疗法文献综述技能

一个用于撰写音乐疗法和音乐干预(MBI)系统性文献综述的综合性 Claude 技能。该技能提供结构化工作流程、模板和指南,用于产出音乐疗法研究领域的高质量学术综述。

[English](README.md) | 中文文档

## 概述

该技能支持音乐疗法研究的系统性和严谨的文献综述写作,涵盖:

* **神经音乐疗法(NMT)**: 节奏听觉刺激(RAS)、旋律语调疗法(MIT)
* **临床应用**: 帕金森病、卒中康复、痴呆、抑郁、焦虑
* **神经机制**: 听觉-运动耦合、奖赏回路、认知通路
* **技术增强干预**: 自适应数字乐器、音乐游戏、AI系统

## 核心特性

### 📚 全面的文献来源

* **PubMed/MEDLINE**: 临床试验和生物医学研究

## 核心原则

### 写作风格

* **限定性语言**: 使用"可能"、"提示"、"似乎"而非绝对断言
* **循证**: 每个主张都需要引用支持
* **精确性**: 区分音乐疗法(需资质)和音乐医学
* **透明性**: 报告效应量、置信区间和局限性

### 必需元素

* 要点框(3-5个要点)
* 每个干预类别的比较表
* 证据质量指标(⬛⬛⬛◻系统)
* 格式一致的效应量
* NIH工具包构建模块合规性
* PRISMA流程图(用于系统评价)

### 术语标准

| 推荐术语        | 避免使用    | 原因       |
| ----------- | ------- | -------- |
| 音乐干预(MBI)   | 音乐治疗    | NIH标准化术语 |
| 音乐疗法        | 音乐疗愈    | 需要专业资质   |
| 节奏听觉刺激(RAS) | 节拍疗法    | 技术精确性    |
| 旋律语调疗法(MIT) | 歌唱疗法    | 既定协议名称   |
| 听觉-运动耦合     | 声音-动作联系 | 神经科学精确性  |

## 工作流程概览

### 阶段1: 项目初始化

* 创建项目结构(CLAUDE.md, IMPLEMENTATION_PLAN.md, manuscript_draft.md)
* 设置写作指南和术语标准

### 阶段2: 文献收集

* 搜索多个数据库(PubMed, Cochrane, PsycINFO, ArXiv)
* 使用PRISMA指南筛选和选择研究
* 进行质量评估
* 创建文献矩阵

### 阶段3: 大纲开发

* 定义理论框架
* 组织干预类别
* 将论文映射到章节
* 规划比较表和图表

### 阶段4: 章节写作

* 撰写理论框架章节
* 使用NIH构建模块撰写干预章节
* 撰写临床应用章节
* 包含局限性和证据质量

### 阶段5: 表格和图表

* 创建比较表
* 生成PRISMA流程图
* 设计神经通路示意图
* 创建证据地图

### 阶段6: 质量保证

* 结构检查
* 内容检查
* NIH工具包合规性验证
* 证据质量检查
* 语言和参考文献检查

### 阶段7: 增量更新

* 设置监控警报
* 季度审查周期
* 版本控制和变更日志
